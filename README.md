# 🌍 Geospatial Query System

A Flask-based geospatial query system that uses NLP and OpenStreetMap to find nearby locations.

## Features

- NLP-based query extraction
- Nearby location search
- Real-time GPS support
- Interactive maps using Folium
- Dynamic place cards
- Voice search support
- Error handling and fallback images

## Technologies Used

- Python
- Flask
- OpenStreetMap
- Nominatim
- Folium
- HTML/CSS/JavaScript
- Bootstrap

## Run Locally

```bash
pip install -r requirements.txt
python app.py
```

## Example Queries

- Hospitals nearby
- Restaurants in Goa
- Tourist places in Jaipur
- Train stations nearby
